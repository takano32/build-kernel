# build-kernel-archlinux

```
$ docker build -t build-kernel-archlinux .
$ docker run -it --rm -p 8000:8000 build-kernel-archlinux
```
Borwse http://localhost:8000/ and you can get built zst packages from `zst-pkg` directory.
